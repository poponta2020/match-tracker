# 練習参加登録画面 - UI改善提案

## 現在の問題点
- 参加予定の人数が表示されていない
- 試合ごとの参加人数が見えない

└────────────────────────────────────────────────────────────────────────────┘

        2025年11月

┌────────────────────────────────────────────────────────────────────────────┐
│ 日付      │ 時間        │ 場所    │ 参加予定 │ 試合1 │ 試合2 │ 試合3 │ ... │
│           │             │         │          │  5名  │  4名  │  6名  │     │
├───────────┼─────────────┼─────────┼──────────┼───────┼───────┼───────┼─────┤
│ 11/15(金) │ 19:00~21:00 │ 体育館A │ 8/20名   │  □   │  □   │  □   │     │
│           │             │         │          │       │       │       │     │
├───────────┼─────────────┼─────────┼──────────┼───────┼───────┼───────┼─────┤

## 提案2: チェックボックスの下に人数を表示

│           │             │         │          │  (5名)  │  (4名)  │  (6名)  │
├───────────┼─────────────┼─────────┼──────────┼─────────┼─────────┼─────────┤
│ 11/18(月) │ 19:00~21:00 │ 体育館B │ 10/25名  │   ☑    │   ☑    │   □    │

## 提案4: バッジ表示（おすすめ）

```
┌────────────────────────────────────────────────────────────────────────────┐
│ 日付      │ 時間        │ 場所    │ 参加予定 │ 試合1 │ 試合2 │ 試合3 │ ... │
├───────────┼─────────────┼─────────┼──────────┼───────┼───────┼───────┼─────┤
│ 11/15(金) │ 19:00~21:00 │ 体育館A │ 8/20名   │  □   │  □   │  □   │     │
│           │             │         │          │  [5]  │  [4]  │  [6]  │     │
├───────────┼─────────────┼─────────┼──────────┼───────┼───────┼───────┼─────┤
│ 11/18(月) │ 19:00~21:00 │ 体育館B │ 10/25名  │  ☑   │  ☑   │  □   │     │
│           │             │         │          │  [7]  │  [8]  │  [5]  │     │
└───────────┴─────────────┴─────────┴──────────┴───────┴───────┴───────┴─────┘

[5] = 小さなバッジ（背景色: グレー、文字: 人数）
```

**メリット:**
- コンパクトで見やすい
- チェックボックスの近くに表示されるため直感的
- 色分けで人数の多寡を表現可能（例: 少ない=赤、普通=グレー、多い=緑）

**デメリット:**
- チェックボックスとの配置調整が必要

## 推奨案: **提案4（バッジ表示）**

最もバランスが良く、以下の理由でおすすめです:

1. **視認性が高い**: 試合ごとの参加人数が一目で分かる
2. **コンパクト**: 既存のレイアウトを大きく変えずに実装可能
3. **直感的**: チェックボックスのすぐ下に人数が表示される
4. **拡張性**: 色分けで混雑度を表現できる
   - 少ない（3名以下）: 赤系のバッジ
   - 普通（4-6名）: グレー系のバッジ
   - 多い（7名以上）: 緑系のバッジ
