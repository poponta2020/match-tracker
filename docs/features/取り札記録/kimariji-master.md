# 決まり字マスター（100枚・確定版）

取り札記録機能のチップ表示に使う、札番号 → 決まり字 の確定マスター。
実装時にフロントの定数（例 `karuta-tracker-ui/src/data/kimariji.js`）へ落とす。

## 表記ルール
- 出典: `docs/reference/決まり字・札番号一覧.txt`（決まり字＝赤字＝text-danger）。
- 共札（長い共通字を持つ4ペア）は **「共通字・区別字」** 記法で **最大4文字**に収める:
  - わた・や(011) / わた・こ(076)、きみ・は(015) / きみ・お(050)、よの・よ(083) / よの・は(093)
  - **あさぼ系は「・」を入れず4文字**: あさぼあ(031) / あさぼう(064)
- 参照ファイルの誤りを補正: 041=こひ（欠落補完）、068=こころに（「こい」重複を訂正）、082=おも（「072」誤記を訂正）。
