## 実装タスク

### タスク1: 凡例UI＋初回自動表示の実装
- [x] 完了
- **概要:** 練習カレンダーに「ⓘ 記号の見方」ボタンと凡例パネルを追加し、初回訪問時の自動表示を実装する。
- **詳細:**
  - カレンダーテーブル直上・右寄せに「ⓘ 記号の見方」ボタン（`lucide-react` の `Info`/`HelpCircle` ＋テキスト、`aria-label="記号の見方を開く"`）を配置。
  - タップで凡例パネルを開閉。既存 `YearMonthPicker`（L615）と同じ**アンカー型ドロップダウン**で、**画面外タップで閉じる**。
  - パネルは**2グループに分離**:
    - ▎試合の空き状況（試合ごと）: ○ 空きあり（緑）／△ 残りわずか（オレンジ）／× 満員（赤）
    - ▎あなたの参加状況: ■ 参加確定（緑系スウォッチ）／■ キャンセル待ち（黄系スウォッチ）
    - 小さく「※試合ごとの空き状況です」を添える。
  - 配色はカレンダー本体に揃える: 記号は `symbolFor`（L721-725）と同色、参加状況スウォッチはセル背景色（confirmed `bg-[#e8efea]`/`border-[#a3c4ad]`、waitlisted `bg-[#fefcf5]`/`border-[#e8d48b]`）。
  - **初回自動表示**: `showLegend` state を `localStorage` キー `practiceCalendarLegendSeen` で初期化（未設定なら自動で開く）。自動表示時に既読フラグを保存。`localStorage` 例外は try/catch でガード。
  - 実装は `PracticeList.jsx` 内のインライン JSX もしくは同ファイル内サブコンポーネント（**汎用化しない**）。
- **変更対象ファイル:**
  - `karuta-tracker-ui/src/pages/practice/PracticeList.jsx` — 凡例ボタン・パネル・`showLegend` state・初回自動表示ロジックを追加（**追加のみ**、既存セル描画は不変）
- **依存タスク:** なし
- **完了条件:**
  - 凡例ボタンが常時表示される（セッション有無に関わらず）
  - タップで開閉でき、画面外タップで閉じる
  - 初回訪問時のみ自動で開く（`localStorage` 既設定時は開かない）
  - 凡例の配色がカレンダー本体と一致
  - 既存のセル描画・グリッド・背景色ロジックが**変化していない**
- **対応Issue:** #934
