# リリースノート（アーカイブ）

※ このファイルには今後追記しない。変更履歴は git と docs/features/ が正。

#### 試合フォーム リデザイン（2026-06-30）
- **MatchForm（1試合入力）を Anti-Slop 階層デザインへ刷新**（design-spec: locked）
  - ヘッダー＝日付(曜)＋会場名／タブ「N試合目」／結果トグル（選択時のみ文字色）／枚数差・お手付きの下線ピッカー（数値＋単位を中央密着）／アクション行「キャンセル｜登録」
  - 対戦相手＝当日参加者（自分を除く）＋「抜け番」のプルダウン＋「未参加から検索」モーダル、相手の級を `(A)` で控えめ表示、抜け番はプルダウン一本化（旧ボタン廃止）
- **未参加相手の自動参加登録**: 詳細登録（`MatchService.createMatch`）時に未参加の対戦選手をサーバ側で `WON` 登録（冪等・同一トランザクション・伝助同期は次回に委譲）
- **日付＋会場ヘッダーの横展開**: `MatchResultsView` / `BulkResultInput` のヘッダーにも会場名を表示（3画面共通）
- 級の短縮表示ヘルパー `utils/rank.kyuRankShortLabel`（"A級"→"(A)"）を追加

#### v2.0.0（2026-03-23）
- **会場管理機能**
  - 会場マスタ（名前、デフォルト試合数）
  - 会場別試合時間割
  - 練習セッションとの連携（venue_id）
- **抽選・キャンセル待ち機能**
  - 定員超過時の自動抽選
  - キャンセル待ち（番号順管理）
  - 繰り上げ通知（応答期限付き）
  - 管理者による参加者手動編集
  - 抽選実行履歴
- **アプリ内通知**
  - 抽選結果通知（当選/落選）
  - 繰り上げ連絡・期限切れ通知
  - 未読バッジ表示
- **Web Push通知（VAPID認証）**
  - ブラウザプッシュ通知対応
  - サブスクリプション管理
- **カレンダー購読（iCalフィード）**
  - 公開URLによる購読型（OAuth不要、全メンバー利用可）
  - 表示名カスタマイズ（プレイヤー×団体ごと）
- **伝助連携**
  - 出欠情報スクレイピング・インポート
- **技術スタック更新**
  - MySQL → PostgreSQL 16（Render.comホスティング）
  - React 18 → 19、React Router v6 → v7
  - Recharts、Lucide React追加
- **新規画面**
  - ランディングページ、プライバシーポリシー、利用規約
  - 試合結果表示、一括試合結果入力
  - 会場一覧・登録、抽選結果、繰り上げ応答、キャンセル待ち状況
  - 通知一覧
- **CI/CD**
  - GitHub Actions（JUnit + Jacoco、最低カバレッジ60%）
  - Render.comデプロイ設定
