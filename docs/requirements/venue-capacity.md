## 1. 概要

会場（Venue）エンティティに定員（capacity）フィールドを追加し、会場ごとに参加可能な最大選手数を登録・管理できるようにする。

### 1.1 背景・目的

- 現在、定員はPracticeSession（練習会）ごとに手動設定している
- 同じ会場であれば定員は基本的に同じであり、毎回入力するのは手間である
- 会場に定員を持たせることで、PracticeSession作成時にデフォルト値として自動入力できるようにする

### 1.2 スコープ

- 会場エンティティへの定員フィールド追加
- 会場登録・編集画面での定員入力対応
- 会場一覧画面での定員表示
- PracticeSession作成時の定員デフォルト値連携

### 1.3 スコープ外

- PracticeSession.capacityの廃止（引き続き練習会ごとに上書き可能なまま維持）
- 抽選機能・キャンセル待ち通知機能の変更（PracticeSession.capacityを参照しており影響なし）

## 2. 用語定義

| 用語 | 定義 |
|------|------|
| 会場 | 練習が行われる場所。Venueエンティティで管理 |
| 定員 | 会場で練習に参加可能な最大選手数 |
| デフォルト定員 | 会場に設定された定員。PracticeSession作成時の初期値として使用 |
| 練習会定員 | PracticeSessionに設定される定員。会場のデフォルト定員を上書き可能 |

## 3. データ仕様

### 3.1 追加フィールド

| 項目 | 値 |
|------|-----|
| エンティティ | Venue |
| フィールド名（Java） | `capacity` |
| DB列名 | `capacity` |
| 型 | Integer |
| NULL許可 | Yes（任意項目） |
| 最小値 | 1 |
| 最大値 | 制限なし |
| デフォルト値 | NULL（未設定） |

### 3.2 DBスキーマ変更

```sql
ALTER TABLE venues ADD COLUMN capacity INT NULL;
```

---

## 4. 機能仕様

### 4.1 会場登録・編集

| 項目 | 仕様 |
|------|------|
| 入力フィールド | 数値入力（任意項目） |
| ラベル | 「定員」 |
| バリデーション | 入力時: 1以上の整数 |
| 未入力時の動作 | NULLとして保存 |

### 4.2 会場一覧表示

| 項目 | 仕様 |
|------|------|
| 表示内容 | 定員数（例:「20人」） |
| 未設定時の表示 | 「未設定」 |

### 4.3 PracticeSession作成時のデフォルト値連携

```
[会場選択] → 会場にcapacityが設定されている?
  Yes → PracticeSessionの定員フィールドにその値を自動入力
  No  → PracticeSessionの定員フィールドは空のまま
```

| 項目 | 仕様 |
|------|------|
| トリガー | PracticeSession作成画面で会場を選択（または変更）した時 |
| 動作 | 選択された会場のcapacityをPracticeSessionの定員フィールドにセット |
| 会場の定員が未設定の場合 | 定員フィールドは空のまま（既に入力済みの値はクリアしない） |
| ユーザーによる変更 | デフォルト値を自由に上書き可能 |

### 4.4 PracticeSession編集時の動作

| 項目 | 仕様 |
|------|------|
| 動作 | 既に保存されているPracticeSessionのcapacityを表示 |
| 会場変更時 | 会場のデフォルト値で**上書きしない**（既存の値を維持） |
| 理由 | 編集時は既に確定した定員が設定されており、意図しない変更を防ぐため |

## 6. 既存機能への影響

| 機能 | 影響 |
|------|------|
| PracticeSession.capacity | **変更なし** — 引き続き練習会単位の定員として使用 |
| 抽選機能（LotteryService） | **変更なし** — PracticeSession.capacityを参照しており、Venue.capacityは参照しない |
| キャンセル待ち通知 | **変更なし** — 同上 |
| 組み合わせ作成（Pairing） | **変更なし** — 定員とは独立した機能 |
| 会場の試合時間割（VenueMatchSchedule） | **変更なし** |
