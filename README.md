## Claude Codeの使用方法

このプロジェクトでは、Cursorエディタ上でClaude Codeを使用できます。

### セットアップ

1. Claude Codeは既にインストール済みです（バージョン: 2.0.29）

2. Cursorとの連携を行うには、ターミナルで以下のコマンドを実行します：
   ```bash
   claude /ide
   ```

3. 初回起動時は、ログインや初期設定が必要な場合があります。

### 使用方法

- Cursorの統合ターミナルで`claude`コマンドを実行してClaude Codeを起動
- `/ide`コマンドでCursorとの連携を確認
- コードの編集や質問などの機能が利用可能になります

### 設定ファイル

`.vscode/settings.json`にClaude Codeの設定が含まれています。
